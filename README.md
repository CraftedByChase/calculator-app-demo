# Calculator App Demo

Description
This is a personal project I made to test my capability at applying simple JavaScript to a website. The calculator can handle the basic four operations as well as conversions between negative and positive numbers.

## Stand-out Features
It can handle multiple operations in an expression and does basic order of operations. It will not calculate if there is an operator but not another number after it. It also will not start with an operator if the output was just cleared.

## Limitations/Issues
The number of operations is very limited. Output can overflow from the box and appear outside of the calculator.
